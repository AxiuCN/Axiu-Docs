# 面板操作

MCSManager 面板的基本使用说明。

## 面板结构

MCSManager 分为两个核心组件：

- **Web 面板**：用户操作界面，管理实例和文件
- **Daemon 守护进程**：运行在节点上，负责启停实例

## 实例管理

### 创建实例

1. 登录面板后进入「应用实例」
2. 点击「新建实例」
3. 选择实例类型（Minecraft 服务端 / 自定义程序）
4. 配置启动命令和工作目录
5. 保存并启动

### 文件管理

面板内置文件管理器，支持：

- 文件上传 / 下载
- 在线编辑文本文件
- 压缩 / 解压

### 终端

每个实例都有独立的终端，可直接在浏览器中操作。

## 节点管理

节点是运行实例的物理机器。一台面板可以管理多个节点。

将 Daemon 安装到新机器后，在面板中添加节点即可统一管理。
