# xitian-plugin

> 源码仓库：<https://github.com/yoimiya-kokomi/xitian-plugin>

### 分组

- **描述**：提供js插件分组管理
- **事件**：`message`
- **优先级**：5

| 正则表达式 | 处理函数 |
|-----------|---------|
| `^#分组列表$` | list |
| `^#创建分组(.\*)$` | new |
| `^#删除分组(.\*)$` | del |
| `^#(.\*)设置分组(.\*)$` | set |
| `^#同步分组$` | sync |

### 插件帮助

- **描述**：发送插件管理器使用帮助
- **事件**：`message`
- **优先级**：5

| 正则表达式 | 处理函数 |
|-----------|---------|
| `^#插件(管理器)?(菜单\|帮助\|help)$` | help |

### 插件安装

- **描述**：进行安装和新增插件操作
- **事件**：`message`
- **优先级**：5

| 正则表达式 | 处理函数 |
|-----------|---------|
| `^#(安装\|新增\|增加)插件(https://(github\|gitee).com/\[a-zA-Z0-9-\]\{1,39\}/\[a-zA-Z0-9\_-\]\{1,100\}(.git)?)?$` | new |
| `^#开始批量安装插件$` | batch |
| `^#清空回收站` | clear |
