# LinkFlow-Plugin

**插件：** BiliLivePush / [LinkFlow]链接解析 / [LinkFlow]帮助 / [LinkFlow]激励计划 / [LinkFlow]账号登录 / [LinkFlow]订阅管理

> 源码仓库：<https://github.com/yoimiya-kokomi/LinkFlow-Plugin>

**共 5 条指令：**

- `^#?(全体|匿名)?订阅直播`
- `^#?(全体|匿名)?取消订阅直播`
- `^#?(全体|匿名)?订阅UP`
- `^#?(全体|匿名)?取消订阅UP`
- `^#?(本?群|我的?)?订阅(列表|list)?`
