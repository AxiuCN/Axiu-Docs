# GamePush-Plugin

> 源码仓库：<https://github.com/yoimiya-kokomi/GamePush-Plugin>

### [GamePush-Plugin]原神功能

- **描述**：原神版本更新及预下载推送
- **事件**：`message`
- **优先级**：99

| 正则表达式 | 处理函数 |
|-----------|---------|
| `^#\*(ys\|YS\|原神)?版本监控$` | ysCheck |
| `^#\*(ys\|YS\|原神)?(开启\|关闭)版本推送$` | ysPushSet |
| `^#\*(ys\|YS\|原神)?当前版本$` | ysVer |
| `^#\*(ys\|YS\|原神)?版本数据(.\*)$` | ysVersionData |
